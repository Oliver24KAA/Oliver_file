"Year-End Equipment Inventory"

Inventory: Chairs, Tables, Equipment in a Room

Guidelines for Filling Up the Requirements:
  • Teacher Name: Enter full name of the responsible teacher (Firstname MI. Surname).
  • Quantity: Number of items available.
  • Articles/Description: Describe the item clearly.
  • Date Acquired: Use format YYYY-MM-DD.
  • Status: Current condition (e.g., Good, Damaged).
  • Unit Price: Price per unit in numeric format.
  • Location: Where the item is located.
  • Classification: Category or type of the item.
